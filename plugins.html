Music Player by Evan Jesty

The music player plays music from any youtube-dl supported site in voice channels. Please read the readme on the projects github page for configuration details.

Commands:

  • play [url] - Plays a youtube url
  • play [search_query] - Searches youtube and plays the first result for the search query
  • playurl [url] - Plays any youtube-dl supported url
  • disconnect - Stop playing and leave the voice channel
  • join [channel] - Join the specified voice channel
  • summon - Join the calling user's voice channel
  • vol [volume] - Sets the volume for the voice player, from 0-100
  • playlist - Display the playlist
  • playlist remove [index] - Remove the item from the playlist
  • playlist move [index] [newindex] - Moves the item from the specified index to the new index
  • playlist inspect [index] - Inspect the item at the specified index
Plugin Manager by Evan Jesty and Ryan Ly

The plugin manager allows you to load, unload and reload plugins dynamically.

Commands:

  • loadplugin [plugin_name] - Loads plugin from the module plugin_name; if the plugin is already loaded it will reload it
  • unloadplugin [plugin_name] - Unloads plugin who's module name is plugin_name
  • listplugins - Lists all loaded plugins
  • enable_command [plugin_name] [command_name] - Enables the command from plugin_name
  • disable_command [command_name] - Disables the command
  • stop - Stops the bot

Ping List by Ryan Ly

Defines a list of users to be pinged at the same time
(e.g. to ping all players of a specific game)

  • ping list - Lists all ping lists
  • ping [list] - Pings all user in the list
  • ping add [list] - Creates a new ping list
  • ping remove [list] - Removes a ping list
  • ping [list] list - Lists all users in list
  • ping [list] add [users] - Adds users to list
  • ping [list] remove [users] - Removes user from list
Admin Utilities by Ryan Ly

Some Utilities that aid in managing a discord server

Autorole - Defines roles to be added on user join

  • autorole list - Lists all autoroles
  • autorole add [autorole_name] - Adds an autorole named role_name
  • autorole remove [autorole_name] - Removes the specified autorole

Purge - Deletes messages

purge [user (optional)] [number] - Purses number messages. If user is defined, only messages from user will be deleted.
